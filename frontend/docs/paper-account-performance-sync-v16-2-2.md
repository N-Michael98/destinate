# V16.2.2 Account to Performance Sync

## Goal

Convert V16 paper account and paper history into performance metrics.

## Input

- paper-position-account-sync
- PaperHistory

## Output

- return percent
- win rate
- loss rate
- profit factor
- average PnL
- best/worst PnL
- performance score
- risk grade

## Safety

No live trading.
No broker execution.
No real orders.
Paper/simulation only.
