# V15.B.7 Dashboard Visual Polish

## Goal

Improve the visual quality of the dashboard navigation without reintroducing emoji or encoding risk.

## What changed

- SidebarIcon now supports active and inactive states
- Active sidebar icons receive cyan highlight styling
- Inactive sidebar icons receive hover styling
- Sidebar button now uses group hover behavior

## Safety

No API changed.
No engine changed.
No trading logic changed.
No Unicode emoji restored.
Only visual UI styling changed.
